# ReadParameterStruct

Template package for reading and storing data from input file to a generic parameters struct in c/cpp.

## Dependencies
- CMake and build essential:
`$ sudo apt install git cmake build-essential`

- [cJSON](https://github.com/DaveGamble/cJSON.git) (Done automatically through Cmake)

## Building

1. After cloning the repository, create a build folder in the root of the package

`$ mkdir build`

2. Configure and compile the code

`$ cd build`

`$ cmake ..`

`$ make`
