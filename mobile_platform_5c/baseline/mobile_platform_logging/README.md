# mobile-platform-logging

## Description
Logging activity for the mobile platform. Initially only for logging KELO data, in the future functionality will be expanded to also allow logging of other hardware.

## Dependencies

### cJSON

#### Download
On Ubuntu systems (tested on Ubuntu 24.01):

`git clone https://github.com/DaveGamble/cJSON.git`

#### Compilation

Navigate to the place where you have cloned this repositoy and create a build folder:

`$ mkdir build `

Enter the build folder and compile the project. 

`$ cd build `

`$ cmake ..`  

`$ make`

If you wish to install the library:

`$ make install`
